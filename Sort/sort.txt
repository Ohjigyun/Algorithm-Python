알고리즘    평균 시간 복잡도    공간 복잡도     특징

선택 정렬       O(N^2)             O(N)       아이디어가 매우 간단하다.

삽입 정렬       O(N^2)             O(N)       데이터가 거의 정렬되어 있을 때 빠르다.

퀵 정렬         O(NlogN)           O(N)       대부분의 경우 가장 적합, 충분히 빠르다

계수 정렬       O(N + K)           O(N+K)     데이터의 크기가 한정되어있을 때 사용 가능, 매우 빠르다.
